This project ONLY works reliably on:
- Raspberry Pi OS Legacy (Bullseye)
- Python 3.9
- OpenCV 4.5.1.48
- NumPy 1.19.5

Bookworm + Python 3.11 WILL NOT WORK.

--------------------------------------
Fresh Install Steps
--------------------------------------
1. Flash Raspberry Pi OS Legacy (Bullseye) Lite
2. Boot Pi and SSH in:
   ssh pi@raspberrypi.local
3. Clone repo:
   git clone -b DEV https://github.com/MangiafestoElectronicsLLC/ME_CAM.git
4. Create Python 3.9 venv:
   python3.9 -m venv venv
   source venv/bin/activate
5. Install dependencies:
   pip install -r requirements.txt
6. Run:
   python3 main.py

--------------------------------------
Run on Boot
--------------------------------------
sudo systemctl enable mecamera
sudo systemctl start mecamera

--------------------------------------
Useful Commands
--------------------------------------
Check service:
systemctl status mecamera

View logs:
journalctl -u mecamera -f

--------------------------------------
Reset Everything
--------------------------------------
rm -rf ~/ME_CAM
rm -rf ~/ME_CAM/venv
sudo apt autoremove -y
sudo apt clean
