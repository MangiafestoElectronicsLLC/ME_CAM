ME CAMERA – PI ZERO 2 W SURVIVAL NOTES
======================================

This file explains how to set up ME Camera from a fresh SD card flash.

--------------------------------------
1. Flash Raspberry Pi OS (Bullseye or Bookworm)
--------------------------------------
Use Raspberry Pi Imager:
- Choose Raspberry Pi OS Lite (32-bit)
- Enable SSH
- Set WiFi + password
- Set username/password
- Flash SD card

Insert SD → boot Pi.

--------------------------------------
2. Connect via SSH
--------------------------------------
ssh pi@raspberrypi.local

--------------------------------------
3. Clone the ME Camera GitHub repo
--------------------------------------
cd ~
git clone -b DEV https://github.com/MangiafestoElectronicsLLC/ME_CAM.git
cd ME_CAM

--------------------------------------
4. Python 3.9 is REQUIRED on Pi Zero 2
--------------------------------------
Python 3.11 cannot install numpy or opencv.
Python 3.9 works perfectly.

If Python 3.9 is not installed:
sudo apt install python3.9 python3.9-venv

--------------------------------------
5. Create the Python 3.9 virtual environment
--------------------------------------
cd ~/ME_CAM
python3.9 -m venv venv
source venv/bin/activate

--------------------------------------
6. Install dependencies
--------------------------------------
pip install --upgrade pip
pip install -r requirements.txt

--------------------------------------
7. Run ME Camera manually
--------------------------------------
source venv/bin/activate
python3 main.py

Open dashboard:
http://raspberrypi.local:8080

--------------------------------------
8. Run ME Camera on boot (systemd)
--------------------------------------
sudo nano /etc/systemd/system/mecamera.service
(paste the service file)

Enable:
sudo systemctl daemon-reload
sudo systemctl enable mecamera
sudo systemctl start mecamera

--------------------------------------
9. Folder Structure
--------------------------------------
ME_CAM/
  venv/
  config/
  recordings/
  exports/
  logs/
  main.py
  setup.sh
  requirements.txt
  notes.txt

--------------------------------------
10. Useful Commands
--------------------------------------
Check service:
systemctl status mecamera

Stop service:
sudo systemctl stop mecamera

Start service:
sudo systemctl start mecamera

View logs:
journalctl -u mecamera -f


--------------------------------------
Make executable:

bash
chmod +x setup.sh
Run:

bash
./setup.sh
✅ 3. systemd Service File
Create:
/etc/systemd/system/mecamera.service

[Unit]
Description=ME Camera Service
After=network.target

[Service]
User=pi
WorkingDirectory=/home/pi/ME_CAM
ExecStart=/home/pi/ME_CAM/venv/bin/python3 /home/pi/ME_CAM/main.py
Restart=always

[Install]
WantedBy=multi-user.target

Enable it:

bash
sudo systemctl daemon-reload
sudo systemctl enable mecamera
sudo systemctl start mecamera
Check status:

bash
systemctl status mecamera
